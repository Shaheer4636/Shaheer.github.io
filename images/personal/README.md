# Personal Photos

This folder is for your personal photos. Upload your images here and reference them in your personal blog posts.

## Usage in Blog Posts

To use an image in a post, add the following front matter:

```yaml
header:
  image: /images/personal/your-photo.jpg
  teaser: /images/personal/your-photo.jpg
```

Or embed inline in markdown:

```markdown
![Caption](/images/personal/your-photo.jpg)
```

## Tips

- Use `.jpg` or `.webp` for photos (smaller file size)
- Keep filenames lowercase with hyphens: `trip-to-lahore.jpg`
- Recommended width: 1200px or wider for header images
- Teaser images display best at 16:9 or 4:3 aspect ratio
